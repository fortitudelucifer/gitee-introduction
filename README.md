# Documenting the changes in my notes throughout the process of learning computer science
A real-time synchronized repository within the Obsidian database containing MD, PDF, MP4, images, and other materials for summarizing computer science learning. The project, utilizing WebDev, SimpRead, and the local Obsidian software for data synchronization provides a low-cost solution for data backup and tracking records of note updates, enables seamless real-time database transfer and editing across personal computers and mobile devices. During note modeling, MATLAB was employed.
