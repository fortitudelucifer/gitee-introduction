# Documenting the changes in my notes throughout the process of learning computer science
A real-time synchronized repository within the Obsidian database containing MD, PDF, MP4, images, and other materials for summarizing computer science learning. Utilizing WebDev, Simpread, and the local Obsidian software for data synchronization provides a low-cost solution for data backup and tracking records of note updates. MATLAB was employed during the note modeling process.
